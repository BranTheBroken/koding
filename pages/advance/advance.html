#include<stdio.h>

int find(int *arr,int n,int k){
         for(int i=0;i<n;i++){
         if(arr[i]==k)
         return 1;
     }
     return 0;
}

void main(){
     int n;
     scanf("%d",&n);
     int arr[n];
     for(int i=0;i<n;i++){
         scanf("%d",&arr[i]);
     }
     int k;
     scanf("%d",&k);
     if(find(arr,n,k)){
         printf("Element found");
     }
     else{
         printf("Element not found");
     }
}
#include<stdio.h>

#include is a statement which tells the compiler to insert the contents of stdio at that particular place. stdio.h is the header file for standard input and output. This is useful for getting the input from the user(Keyboard) and output result text to the monitor(screen). With out this header file, one can not display the results to the users on the screen or cannot input the values through the keyboard.

void main()

main() function is the entry point of any C program. It is the point at which execution of program is started. When a C program is executed, the execution control goes directly to the main() function.And void specifies that the function does not return any value.

int n

"int n" means that we want to create a variable or container name 'n' which can store ony integer value.We can name the variable anything but it has some rules,you can look that up in net, and just like int we can use float and double to store floating point or real number and char to store characters,boolean for true and false.

scanf

The scanf function allows you to accept input from standard in, which for us is generally the keyboard. "scanf("%d", &n);"" The program will read in an integer value that the user enters on the keyboard (%d is for integers, as is printf, so b must be declared as an int) and place that value into n.

arr[n]

what "int arr[n]" does that it allocate memory for n elements of type int, that is we can store n values of type int.Each value can be accessed by arr[i] where i is the index of element which varies from 0 to n-1

for loop

"for loop" repeats the statement inside it repeatedly until the condition is false.The first statement inside for loop is initialisation of counter variable "i",second statement is condition ,the statement runs until this condition becomes false.The third statement is to update the counter variable so that loop doesn't become infinite.

if else

"if else" is a conditional statement that which check the condition in the "if statement",and if the condition is true it executes the statement inside if block otherwise the statements in else block.There are other ways also like else if statements,you can check more info on net.

find

find is a function,which takes in three parameters,array,length of array and element to be found,it passes the parameters to function where by using the logic it return an integer value ,1 or 0 depending on whether element is found or not,and 1 and 0 can be treated as true and false hence it is written as if condition.Try to figure out the logic in "find" on your own.
